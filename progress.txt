## Codebase Patterns
- Vite + React 18 + TypeScript 5, module type ESM
- Vitest for testing with jsdom environment, setup file at src/test/setup.ts
- Tailwind CSS v4 via @tailwindcss/vite plugin (no tailwind.config.js needed)
- PixiJS 8 requires class-based mocks in tests (not vi.fn().mockImplementation)
- Types defined in src/types/index.ts
- VITE_SEOUL_API_KEY env var declared in src/vite-env.d.ts
- HotspotLayer renders inside SeoulMap's SVG as children
- SeoulApiService.fetchAreas uses Promise.allSettled (never throws) - isOffline detected by empty result map
- useSeoulData hook tests need 30s+ timeout (fetches 122 areas)
